# Loan Calculator

Простой калькулятор кредитов для расчета ежемесячных платежей по аннуитетной схеме.

## Описание

Этот проект представляет собой веб-приложение для расчета ежемесячного платежа по кредиту. Пользователи могут настроить основные параметры кредита с помощью интуитивных кнопок "+" и "-" и получить точный расчет выплат.
